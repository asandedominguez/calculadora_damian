Pregunta de control 1: ¿Por qué es necesario realizar un "clone" al inicio en lugar de simplemente copiar el archivo manualmente?


Porque al hacer un clone no solo se copia el contenido del proyecto, sino también todo el historial de versiones, las ramas y la configuración del repositorio Git.
